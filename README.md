# @acdvorak/edid-parser

Pure TypeScript EDID parser with zero dependencies. Supports all JS runtimes.

## Development

- Install dependencies:

```bash
npm install
```

- Run the unit tests:

```bash
npm run test
```

- Build the library:

```bash
npm run build
```
